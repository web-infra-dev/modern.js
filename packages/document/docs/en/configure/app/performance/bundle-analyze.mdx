# performance.bundleAnalyze

- **Type:** `Object | undefined`

Used to enable bundle analysis functionality to analyze the size of the build output.

By default, Modern.js does not enable this feature. When this feature is enabled, the default configuration is as follows:

```js
const defaultConfig = {
  analyzerMode: 'static',
  openAnalyzer: false,
  // Distinguish by target names, such as `web`, `node`, etc.
  reportFilename: `report-${target}.html`,
};
```
