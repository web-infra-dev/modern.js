# performance.bundleAnalyze

- **类型：** `Object | undefined`

用于开启构建产物分析功能，分析打包后的代码体积。

默认情况下，Modern.js 不会开启此功能。当开启该功能后，内部的默认配置如下：

```js
const defaultConfig = {
  analyzerMode: 'static',
  openAnalyzer: false,
  // target 为编译目标，如 `web`、`node` 等
  reportFilename: `report-${target}.html`,
};
```
