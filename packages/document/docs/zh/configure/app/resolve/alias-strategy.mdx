# resolve.aliasStrategy

- **类型:** `'prefer-tsconfig' | 'prefer-alias'`
- **默认值:** `'prefer-tsconfig'`

设置路径别名解析的策略，用于控制 `tsconfig.json` 中的 paths 选项与 Rsbuild 的 [resolve.alias](https://v2.rsbuild.dev/config/resolve/alias) 选项之间的优先级关系。
