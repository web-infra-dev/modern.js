# resolve.dedupe

- **类型:** `string[]`

强制 Rsbuild 从项目根目录解析指定的包，这可以用于移除重复包和减少包大小。对应 Rsbuild 的 [resolve.dedupe](https://v2.rsbuild.dev/config/resolve/dedupe) 配置。
