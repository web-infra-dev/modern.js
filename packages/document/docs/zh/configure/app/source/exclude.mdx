# source.exclude

- **类型：** [RuleSetCondition[]](https://rspack.rs/zh/config/module#condition)
- **默认值：** `[]`

指定不需要编译的 JavaScript/TypeScript 文件。用法与 Rspack 中的 [Rule.exclude](https://rspack.rs/zh/config/module#ruleexclude) 一致，支持传入字符串或正则表达式来匹配模块的路径。
