# output.disableSourceMap

- **Type:**

```ts
type DisableSourceMap =
  | boolean
  | {
      js?: boolean;
      css?: boolean;
    };
```

- **Default:**

```ts
const defaultDisableSourceMap = {
  js: false,
  css: process.env.NODE_ENV === 'production',
};
```

Whether to disable Source Map generation.

:::warning
**Deprecated**: This configuration is deprecated, please use the boolean usage of `output.sourceMap` instead.
:::
