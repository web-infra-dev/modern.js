# output.minify

- **Type:**

```ts
type Minify =
  | boolean
  | {
      js?: boolean;
      jsOptions?: SwcJsMinimizerRspackPluginOptions;
      css?: boolean;
      cssOptions?: LightningCssMinimizerRspackPluginOptions;
    };
```

- **Default:** `true`

Configure whether to enable code minification in production mode, or to configure minimizer options.
