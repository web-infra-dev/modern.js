# security.nonce

- **Type:**

```ts
type Nonce = string;
```

- **Default:** `undefined`

Adding a `nonce` attribute to the scripts resources introduced for HTML. This allows the browser to determine whether the script can be executed when it parses inline scripts with matching nonce values.
