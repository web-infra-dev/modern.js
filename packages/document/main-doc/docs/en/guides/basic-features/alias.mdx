## Default Aliases

The Modern.js framework comes with the following aliases built-in:

```json
{
  "@": "./src",
  "@shared": "./shared"
}
```

Additionally, when the BFF plugin of the framework is enabled, the `@api` alias is automatically added.

```json
{
  "@api": "./api"
}
```
