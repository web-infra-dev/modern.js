import { PackageManagerTabs } from '@theme';

1. 执行 `new` 命令：

<PackageManagerTabs command="run new" />

2. 按照提示，选择**启用 BFF 功能**：

```bash
? 请选择你想要的操作 启用可选功能
? 请选择功能名称 启用「BFF」功能
```

3. 将下面的代码添加到 `modern.config.[tj]s` 中：


```ts title="modern.config.ts"
import { bffPlugin } from '@modern-js/plugin-bff';

export default defineConfig({
  plugins: [bffPlugin()],
});
```
