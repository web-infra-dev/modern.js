# dev.watchFiles

- **类型：**

```ts
type WatchFiles =
  | {
      paths: string | string[];
      type?: 'reload-page';
      //  chokidar 选项
      options?: WatchOptions;
    }
  | {
      paths: string | string[];
      type?: 'reload-server';
    };

type WatchFilesConfig = WatchFiles | WatchFiles[];
```

- **默认值：** `undefined`

监听指定文件和目录的变化。当文件发生变化时，可以触发页面的重新加载，或者触发 dev server 重新启动。
