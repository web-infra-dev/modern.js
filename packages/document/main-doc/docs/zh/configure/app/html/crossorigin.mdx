# html.crossorigin

- **类型：** `boolean | 'anonymous' | 'use-credentials'`
- **默认值：** `false`

用于设置 `<script>` 和 `<style>` 标签的 [crossorigin](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin) 属性。
