# html.meta

- **类型：** `Object | Function`
- **默认值：**

```js
const defaultMeta = {
  charset: { charset: 'utf-8' },
  viewport:
    'width=device-width, initial-scale=1.0, shrink-to-fit=no, viewport-fit=cover, minimum-scale=1.0, maximum-scale=1.0, user-scalable=no',
  'http-equiv': { 'http-equiv': 'x-ua-compatible', content: 'ie=edge' },
  renderer: 'webkit',
  layoutmode: 'standard',
  imagemode: 'force',
  'wap-font-scale': 'no',
  'format-detection': 'telephone=no',
};
```

配置 HTML 页面的 `<meta>` 标签。

import RsbuildConfig from '@site-docs/components/rsbuild-config-tooltip';

<RsbuildConfig configName="html.meta" />
