# output.charset

- **类型：** `'ascii' | 'utf8'`
- **默认值：** `'ascii'`

`charset` 配置项允许你指定输出文件的[字符编码](https://developer.mozilla.org/en-US/docs/Glossary/Character_encoding) 字符编码，以确保它们在不同的环境中能够正确显示。
