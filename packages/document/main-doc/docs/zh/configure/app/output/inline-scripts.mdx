# output.inlineScripts

- **类型：**

```ts
type InlineScriptsTest =
  | RegExp
  | ((params: { size: number; name: string }) => boolean);

type InlineScripts =
  | boolean
  | InlineScriptsTest
  | {
      enable?: boolean | 'auto';
      test: InlineScriptsTest;
    };
```

- **默认值：** `false`

用来控制是否用 `<script>` 标签将产物中的 script 文件（.js 文件）inline 到 HTML 中。

:::tip
当使用约定式路由时，如果开启了这个选项，需要将 [output.splitRouteChunks](/configure/app/output/split-route-chunks) 设置为 false。
:::
