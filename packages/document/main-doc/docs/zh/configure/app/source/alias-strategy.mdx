# source.aliasStrategy

- **类型：** `'prefer-tsconfig' | 'prefer-alias'`
- **默认值：** `'prefer-tsconfig'`

`source.aliasStrategy` 用于控制 `tsconfig.json` 中的 `paths` 选项与打包工具的 `alias` 选项的优先级。
