# source.decorators

- **类型：**

```ts
type Decorators = {
  version?: 'legacy' | '2022-03';
};
```

- **默认值：**

```js
const defaultDecorators = {
  version: 'legacy';
};
```

:::warning
该配置项的使用方式与 Rsbuild 一致，但默认值不同。详细信息请参考 [Rsbuild - source.decorators](https://rsbuild.rs/zh/config/source/decorators)。
:::

我们发现大部分的项目仍使用 `legacy` 版本的装饰器语法，因此默认值为 `legacy`。
