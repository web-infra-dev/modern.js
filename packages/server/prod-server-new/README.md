# Modern.js Package

## Get Started

Run and debug the module:

```bash
pnpm run dev
```

Run test cases:

```bash
pnpm run test
```

Build the module for production:

```bash
pnpm run build
```

Enable optional features:

```bash
pnpm run new
```


For more information, see the [Modern.js Module documentation](https://modernjs.dev/module-tools/en).
