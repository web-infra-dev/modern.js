# Modern.js App

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get Started

Start the start server:

```bash
pnpm start
```

Build the app for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm serve
```

For more information, see the [Modern.js documentation](https://modernjs.dev/en).
