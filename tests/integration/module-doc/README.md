# integration-module-doc

## Get Started

按开发环境的要求，运行和调试项目

开启模块工程的调试模式
// FIXME

```
cd packages/solutions/module-tools && pnpm debug-mode
```

文档预览

```
pnpm run dev
```

构建文档站点

```
pnpm run build --platform
```

运行测试用例

```
pnpm run test
```

按照社区规范和最佳实践，生成构建产物

```
pnpm run build
```

继续创建更多项目要素

```
pnpm run new
```
