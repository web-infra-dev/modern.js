# Alert

## Basic
Warning prompts. Suitable for displaying short warning prompts in a way that attracts attention.

```jsx
import React from 'react';
import { AlertTest } from 'demo';

export default () => {
  return <AlertTest content={'Here is demo1'}></AlertTest>;
};

```

## Closable
Use closable = true to turn on the close button

```jsx
import React from 'react';
import { AlertTest } from 'demo';

export default () => {
  return <AlertTest closable={true} content={'Here is demo2 '}></AlertTest>;
};

```
