# Button


## Size
Buttons can be mini, small, medium and large in size

```jsx
import React from 'react';
import { ButtonTest } from 'demo';

export default () => {
  return <ButtonTest size="large">click me</ButtonTest>;
};

```

## Disabled
The disabled state of the button.

```jsx
import React from 'react';
import { ButtonTest } from 'demo';

export default () => {
  return <ButtonTest disabled>click me</ButtonTest>;
};

```
