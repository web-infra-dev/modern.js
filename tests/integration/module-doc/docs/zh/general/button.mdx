# Button


## 基本用法
按钮分为：迷你、小、中、大，四种尺寸。

```jsx
import React from 'react';
import { ButtonTest } from 'demo';

export default () => {
  return <ButtonTest size="large">点我</ButtonTest>;
};

```

## 可禁用
指定 disabled = true，可禁用按钮。

```jsx
import React from 'react';
import { ButtonTest } from 'demo';

export default () => {
  return <ButtonTest disabled>点我</ButtonTest>;
};

```
