# output.favicon



* 类型： `string`
* 默认值： `undefined`

设置页面的 favicon 图标。

例如以下配置：

```js title="modern.config.js"
import { defineConfig } from '@modern-js/app-tools';

export default defineConfig({
  output: {
    favicon: './src/assets/icon.png',
  },
});
```

`dev` 之后可以看到 html `head` 标签内生成如下 `link` 标签：
