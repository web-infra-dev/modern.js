应用工程方案 App Icon 文件。

项目根目录下 `config/icon.png` 时，可以在构建时在 html 页面注入 app icon 信息, 如下:

```
./config
└── icon.png
```

最终 html 中会注入 app icon `link` 标签, 如下:
