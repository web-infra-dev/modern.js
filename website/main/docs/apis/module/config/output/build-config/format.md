# buildConfig.format



* 类型： `'cjs' | 'esm' | 'umd'`
* 默认值： `'cjs'`

设置模块中 JavaScript 产物的输出格式。

```js title="modern.config.js"
import { defineConfig } from '@modern-js/module-tools';

export default defineConfig({
  output: {
    buildConfig: {
      format: 'cjs',
    },
  },
});
```
